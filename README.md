# Jogos 2D Top-Down na Unity

## Descrição
Este repositório contém uma coleção de scripts e recursos para o desenvolvimento de jogos 2D Top-Down na Unity. Os scripts incluem funcionalidades como movimento de personagens, ataques, IA de inimigos, e muito mais.

## Estrutura do Projeto
- `Assets/`: Contém todos os recursos do projeto, incluindo scripts, sprites, animações, e cenas.
  - `Scripts/`: Contém todos os scripts C# utilizados no projeto.
  - `Sprites/`: Contém todos os sprites e imagens utilizados no projeto.
  - `Animations/`: Contém todas as animações utilizadas no projeto.
  - `Scenes/`: Contém todas as cenas do projeto.

## Scripts
### 1. AtaqueInimigo.cs
Este script define o comportamento de um inimigo que pode atacar o jogador quando este entra em sua área de alcance.

### 2. MovimentoInimigo.cs
Este script define o movimento de um inimigo que se desloca entre dois pontos (`pontoa` e `pontob`) e realiza um movimento de flutuação vertical.

### 3. PlayerControle.cs
Este script define o movimento do jogador em um plano 2D, permitindo que o jogador se mova em todas as direções (horizontal e vertical) com base na entrada do usuário.

### 4. SaveData.cs
Este script permite salvar a posição e a vida do jogador quando ele colide com um ponto de salvamento.

### 5. MenuManage.cs
Este script define a funcionalidade de um menu de jogo, permitindo que o jogador inicie o jogo ou saia dele.
